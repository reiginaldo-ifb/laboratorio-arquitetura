# Laboratório Digital de Arquitetura de Computadores

SPA (Single Page Application) desenvolvida para a disciplina de Programação para WEB I - IFB.

## 🚀 Tecnologias Utilizadas
- HTML5 (Semântico)
- CSS3 (Flexbox/Grid/Variables)
- JavaScript Puro (ES6 Modules)

## 🛠️ Requisitos Atendidos
- **Orientação a Objetos:** Classes para Lógica e Conversão.
- **Programação Funcional:** Uso de `.map()` e `.filter()`.
- **AJAX:** Consumo da Dog API para demonstração de integração.
- **Manipulação do DOM:** Interface 100% dinâmica.

## 📖 Como usar
1. Clone o repositório.
2. Abra o `index.html` usando um servidor local (ex: Live Server).
3. Navegue entre as abas para explorar as funcionalidades.
